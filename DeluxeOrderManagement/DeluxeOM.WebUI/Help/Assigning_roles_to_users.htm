Assigning roles to users

Deluxe Avails enables you to assign a role to a user. Only an Administrator can assign a role to a user. Deluxe Avails provides three roles for its users as follows:

  • Administrator

  • Manager

  • Analyst

To assign roles to users:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click USER MANAGEMENT.

The USERS: LIST page is displayed.

3.    On the USERS: LIST page, click corresponding to the user name to which you want to assign a role.

The USERS : EDIT page is displayed.

4.    On the USERS : EDIT page, in the Role field, select any one of the available roles, and then click SAVE.

The selected role is assigned to the user.
