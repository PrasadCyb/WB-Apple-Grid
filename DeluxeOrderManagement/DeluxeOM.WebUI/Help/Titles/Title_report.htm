Title report

A Title report is the data displayed on searching a specific title. You can use the filters to search a title. The Title report can be exported to an MS Excel file.

 

To create a title report:

1.    On the Deluxe Avails application home page, click the TITLE tab.

The FILTERS page is displayed.

2.    On the FILTERS page, provide the field values shown in the following table, and then click APPLY.

The title information is displayed.

Fields

 Action
Content Provider Select the name of the Content Provider.
Content Distributor Select the name of the Content Distributor.
Selected Title Select the title.
Edit Type Select any one of the following edit types:
  • New Release
  • TV New Release
  • Catalog Promo
  • Theatrical New Release
  • Direct to Video New Release
  • Catalog New To Format
  • Franchise
  • Catalog
Territory This field contains a list of the various territory names where WB has released a movie.
Language This field contains a list of the various languages in which WB has released a movie.
Video Version Select any one of the listed video versions.
MPM Select any one of the listed MPM versions. MPM is the number that identifies a title (intellectual property).
Start Date Select the Start Date.
End Date Select the End Date.

  • To clear all fields and perform a new search, click CLEAR ALL.

All the field data is cleared. You can now provide new field values.

  • To export the result, click EXPORT EXCEL.

The displayed result is exported in the form of an MS Excel file.
