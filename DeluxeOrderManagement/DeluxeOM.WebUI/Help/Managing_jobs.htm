Managing jobs

Importing a report into the Deluxe Avails application is termed as a job. You can check the status of a job or run a job. Following are the various jobs available in the Deluxe Avails application.

  • Apple Price Report

  • Apple QC Report

  • Pipeline Orders

  • Announcement

 

Managing a job includes the following two processes:

  • Run a job

  • View a job

 

To run a job:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click JOBS.

The JOBS page is displayed.

3.    On the JOBS page, in the Job List, select a job, and then click RUN.

The selected file is imported into the Deluxe Avails application.

  • To refresh the job list, click REFRESH.

The JOBS result page is refreshed to display the latest jobs that are run.

 

To view a job:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click JOBS.

The JOBS page is displayed.

3.    On the JOBS page, in the Job List, select a job, and then click RUN.

4.    On the JOBS result page, under the Action column, click View corresponding to the job type you want to view.

The job details are displayed.

 

To run an Announcement job for a past date:

  1. From the Job List, select Announcement.

A Calendar icon is displayed adjacent to the job Date.

  1. Select the date of receipt of the Announcement file and click RUN.

The records for the week starting from the selected date are imported into the Deluxe Avails application.
