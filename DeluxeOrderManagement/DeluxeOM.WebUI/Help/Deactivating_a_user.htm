Deactivating a user

You can deactivate a user in Deluxe Avails. Once a user is deactivated, it cannot log on to the Deluxe Avail application. Only an Administrator can deactivate a user.

 

To deactivate a user:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click USER MANAGEMENT.

The USERS: LIST page is displayed.

3.    On the USERS: LIST page, click corresponding to the user name which you want to deactivate.

The USERS : EDIT page is displayed.

4.    On the USERS : EDIT page, deselect the Active field, and then click SAVE.

The user is deactivated successfully.
