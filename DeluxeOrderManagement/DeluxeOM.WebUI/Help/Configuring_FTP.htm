Configuring FTP

You can configure the FTP by providing details such as server name, port, and file sources. FTP is a location where the announcement files and price reports from Apple iTunes are dumped periodically. These files are extracted by the Deluxe Avails application to identify the orders to process. It is mandatory for you to provide the correct source for each file type that resides in the FTP.

 

To configure the FTP:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click CONFIGURE FTP.

The CONFIGURE FTP page is displayed.

3.    On the CONFIGURE FTP page, provide the field values shown in the following table, and then click SAVE.

The FTP details are updated successfully.

Note: The value for FTP Server Name, Username, Password, Port, and Source vary. Contact your network administrator for these values.

Fields

Action
FTP Server Name Type the server name of the FTP.
Username Type the username required to access FTP.
Password Type the password required to access FTP.
Port Type the Port number of the FTP.
Avail Announcement Type the source path for Avail Announcement file.
Apple Price Report Type the source path for Avail Apple Price Report.
Apple QC Report Type the source path for Apple QC Report.
Pipeline Orders Type the source path for Pipeline Orders.
