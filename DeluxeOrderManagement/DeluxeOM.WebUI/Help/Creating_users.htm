Creating users

You can create new users in Deluxe Avails. To create a new user, you must provide information for fields displayed on the USER-CREATE NEW page.

 

To create a user:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click USER MANAGEMENT.

The USERS: LIST page is displayed.

3.    On the USERS: LIST page, click CREATE NEW.

The USERS-CREATE NEW page is displayed.

4.    On the USERS-CREATE NEW page, provide the field values shown in the following table, and then click CREATE.

A new user is created successfully.

Fields

 Action

First Name

 Type the First Name of the user.

Last Name

 Type the Last Name of the user.

Email

 Type the email ID of the user.

Phone No

 Type the phone number of the user. It must contain numeric values.

Active

 Select this check box to make the user active.

Password

 Type the password for the user. The password must be of at least 12 characters, and contain at least one uppercase and one lower case letter, one number, and one special character (!@#$&*?)

Confirm Password

 Retype the same password that you used for the Password field.

Role

 Select any one of the following values:
  • Administrator
  • Manager
  • Analyst

  • To cancel the creation of a new user and to go back to the USERS- CREATE NEW page, click Back to List.
