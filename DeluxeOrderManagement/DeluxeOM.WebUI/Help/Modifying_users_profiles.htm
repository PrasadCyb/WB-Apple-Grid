Modifying users' profiles

An Administrator can modify a user's profile in Deluxe Avails.

 

To modify a user's profile:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click USER MANAGEMENT.

The USERS: LIST page is displayed.

3.    On the USERS: LIST page, click corresponding to the user name whose profile you want to modify.

The USERS : EDIT page is displayed.

4.    On the USERS : EDIT page, modify the user's fields, and then click SAVE.

The user's profile is modified successfully.
