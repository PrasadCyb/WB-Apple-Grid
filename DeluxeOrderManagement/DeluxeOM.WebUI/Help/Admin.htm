Admin

An Administrator is a super-user having all the privileges to work on the Deluxe Avails application. The Administrator can perform various tasks such as configuring the File Transfer Protocol (FTP), setting up alerts, creating users, assigning roles to users.

This section includes:

  • Configuring FTP

  • Setting up alerts

  • Managing jobs

  • Creating users

  • Assigning roles to users

  • Deactivating a user

  • Modifying user's profiles
