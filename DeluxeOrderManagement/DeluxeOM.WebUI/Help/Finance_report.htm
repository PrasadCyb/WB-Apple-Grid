Finance report

The purpose of the Finance report is to send the information to the Billing department. The data contained in this report is cumulative.

 

To generate a Finance report:

1.    On the Deluxe Avails application home page, click the REPORTS tab.

The FILTERS page is displayed.

2.    On the FILTERS page, provide the field values shown in the following table, select Finance Report as the Report Title, and then click EXPORT EXCEL.

An MS Excel file for the Announcement Analysis Report is exported successfully.

Fields

 Action
Content Provider Select the name of the Content Provider.
Content Distributor Select the name of the Content Distributor.
Report Title Select the report title Finance Report.
Import Date Range Select the date range when the record was imported.
