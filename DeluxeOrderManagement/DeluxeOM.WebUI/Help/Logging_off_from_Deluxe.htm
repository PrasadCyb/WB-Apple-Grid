Logging off from Deluxe

You can log off from the Deluxe Avails application. Logging off from the application will land you on the login page of the Deluxe Avails application.

 

To log off from the application:

  • On the Deluxe Avails application home page, point to the user name on the top-right corner of the screen, and then click LOG OFF.

You are now logged out of the Deluxe Avails application.
