Setting up alerts

The Deluxe Avails application enables you to setup alerts for essential activities, that are performed during the order management process. These activities include change in import status, creation of new title, change in title list, and so on. You can either enable or disable alert notifications for each activity. An email is sent to all the users, who have opted to receive notification alerts.

 

Following are the events for which you can set up alerts.

  • Apple Price Report Import Status

  • Apple QC Report Import Status

  • Avail Announcement File Import Status

  • Pipeline Orders File Import Status

  • New Title in Avail Announcement

  • Changes in any of the actionable columns of the Avail Announcement (Client Start, Client End, Client Avail Status, Local Edit Required, Local Data - Edit EIDR)

 

To set up alerts:

1.    On the Deluxe Avails application home page, click the ADMIN tab.

A list of options is displayed.

2.    On the options list, click NOTIFICATION MAPPING.

The NOTIFICATION MAPPING page is displayed.

3.    On the NOTIFICATION MAPPING page, select the Notification Enable/Disable check box corresponding to the event name, for which you want to set an alert.

4.    In the Email box, type the email IDs of the users separated by a comma.

An email is sent to each of the users when the associated event occurs.

5.    Click SAVE to save the notification changes.
