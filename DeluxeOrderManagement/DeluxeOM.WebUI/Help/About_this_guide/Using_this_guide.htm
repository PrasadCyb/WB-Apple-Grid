Using this guide

This guide is organized as follows.

Chapter Brief description
Introduction

Provides an overview of the Deluxe Avails application, and steps for logging on to the application and changing or retrieving password

Order

Provides steps for all aspects of order management that include creating a new order, searching for an existing order, modifying the existing order details, and exporting the order data into an MS Excel file
Titles

Provides steps to perform store checks, region reporting, research on titles assets and backfill projects.
Reports

Provides information on all types of reports used within the application, such as Announcement analysis report, Finance report, and Cancellation report
Admin

Provides information regarding all activities of an Administrator, such as configuring FTP, setting alerts, managing jobs, creating users, searching users, assigning roles to users, modifying user profiles, and deactivating a user
FAQ

Provides a list of frequently asked questions.
