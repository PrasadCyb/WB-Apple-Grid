Announcement analysis report

The Announcement analysis report contains data of new titles, titles launched in new territories, and titles launched in new languages. The data from the Announcement analysis report is used for creating new orders in TOPs. TOPs is one of the order management system used by Deluxe to raise an order.

 

To generate an Announcement Analysis report:

1.    On the Deluxe Avails application home page, click the REPORTS tab.

The FILTERS page is displayed.

2.    On the FILTERS page, provide the field values shown in the following table, select Announcement Analysis Report as the Report Title, and then click EXPORT EXCEL.

An MS Excel file for the Announcement analysis report is exported successfully.

Fields

 Action
Content Provider Select the name of the Content Provider.
Content Distributor Select the name of the Content Distributor.
Report Title Select the Report Title Announcement Analysis Report.

Video Version

 Type the video version.
Local Edit Select one of the following values:
  • Yes
  • No
Record Created Date Range Select the date range when the record was created.
