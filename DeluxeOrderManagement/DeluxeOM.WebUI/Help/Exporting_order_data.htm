Exporting order data

After searching for orders, you can export the order data into an MS Excel file, which contains all the information required to successfully place an order. Once an order is placed, the order data is sent back to Deluxe Media, for enriching the order data within Deluxe Avails.

 

To export an order data:

1.    On the Deluxe Avails application home page, click the ORDER MANAGEMENT tab.

The FILTERS page is displayed.

2.    On the FILTERS page, type the field values, and then click APPLY.

The order results are displayed.

3.    On the order results page, click EXPORT EXCEL.

An MS Excel file containing the order data is downloaded in the system.
